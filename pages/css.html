Le CSS

"La présentation du site."

Le CSS c'est quoi ?

Le CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire l'apparence et le formatage d'un document HTML. Il permet de contrôler les couleurs, les polices, les espacements et bien d'autres aspects visuels des pages web. Le CSS fonctionne en ciblant des éléments HTML à l'aide de sélecteurs et en leur appliquant des propriétés et des valeurs.

Voici un exemple de base en CSS : 


                        /* Exemple de règles CSS de base */
                        body {
                            font-family: Arial, sans-serif;
                            background-color: #f4f4f4;
                        }
                        
                        h1 {
                            color: #333;
                        }

Bonnes pratiques en CSS

  • Utiliser des sélecteurs spécifiques : Privilégiez les sélecteurs de classe et d'identifiant pour cibler des éléments spécifiques plutôt que d'utiliser des sélecteurs globaux qui peuvent affecter d'autres éléments.
  • Organiser le code CSS : Structurez vos fichiers CSS en sections logiques et commentez votre code pour faciliter la maintenance.
  • Utiliser des unités relatives : Privilégiez les unités comme em ou rem pour les tailles de police et les espacements afin d'améliorer la réactivité.
  • Minimiser l'utilisation d'ID : Évitez d'utiliser des identifiants sauf si nécessaire, car ils sont trop spécifiques et limitent la réutilisation.
  • Valider votre code CSS : Utilisez des outils en ligne pour vérifier la validité de votre CSS et éviter les erreurs.

Accessibilité en CSS

L'accessibilité est essentielle pour garantir que votre site soit agréable à utiliser pour tous. Voici quelques pratiques recommandées :

  • Contraste des couleurs : Assurez-vous que le contraste entre le texte et l'arrière-plan est suffisant pour les utilisateurs malvoyants.
  • Utiliser des classes CSS pour les styles : Évitez d'utiliser des styles en ligne afin de garder votre HTML propre et accessible.

Gestion des médias en CSS

  • Utiliser les media queries : Les media queries permettent de créer des styles réactifs qui s'adaptent à différentes tailles d'écran.
  • Optimisation des images : Utilisez des classes CSS pour gérer la taille et l'affichage des images pour garantir une bonne performance.

Exemple de gestion des médias en CSS :


                        /* Media query pour les écrans plus petits */
                        @media (max-width: 600px) {
                            body {
                                font-size: 14px;
                            }
                        }
                        
                        /* Style pour les images */
                        img {
                            max-width: 100%;
                            height: auto;
                        }
