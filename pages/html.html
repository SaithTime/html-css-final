Le HTML

"La structure du site."

Le HTML c'est quoi ?

Le HTML (HyperText Markup Language) est le langage standard utilisé pour structurer le contenu des pages web. Il permet de définir des titres, des paragraphes, des images, des liens et bien plus encore. Il fonctionne en organisant des éléments appelés balises, qui sont souvent imbriquées les unes dans les autres, comme des "boîtes dans des boîtes".

Voici un exemple de base en HTML : 


                        <!DOCTYPE html>
                        <html lang="fr">
                          <head>
                            <title>Mon site web</title>
                          </head>
                          <body>
                            <h1>Bonjour le monde</h1>
                          </body>
                        </html>

Exemple des bonne pratique: 


                        <!DOCTYPE html>
                        <html lang="fr">
                          <head>
                            <meta charset="UTF-8"> 
                            <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
                            <title>Exemple de bonnes pratiques HTML</title>
                          </head>
                          <body>
                            <header>
                              <h1>Bienvenue sur mon site</h1>
                            </header>
                        
                            <main>
                              <article>
                                <h2>Pourquoi utiliser les balises sémantiques ?</h2>
                                <p>Les balises sémantiques aident à structurer correctement le contenu.</p>
                                <img src="image.jpg" alt="Description de l'image" /> 
                              </article>
                            </main>
                        
                            <footer>
                              <p>© 2024 Mon Site Web</p>
                            </footer>
                          </body>
                        </html>

Bonnes pratiques en HTML

  • Utiliser des balises sémantiques : Préférez des balises comme <header>, <main>, <footer> plutôt que des <div> génériques. Cela améliore la lisibilité du code et l'accessibilité.
  • Bien structurer le code : Organisez votre HTML en utilisant l'indentation correcte pour que le code soit lisible et maintenable.
  • Utiliser des attributs alt pour les images : Ajoutez toujours une description alternative avec l'attribut alt pour les images afin de rendre la page accessible aux personnes utilisant des lecteurs d'écran.
  • Minimiser les répétitions : Utilisez les classes CSS ou les identifiants id pour appliquer des styles communs à plusieurs éléments, au lieu de répéter le code.
  • Valider votre code : Utilisez des outils comme le validateur W3C pour vérifier que votre HTML est conforme aux standards.

Accessibilité en HTML

L'accessibilité est un aspect crucial du développement web, garantissant que votre site soit utilisable par tous, y compris les personnes en situation de handicap.

  • Utiliser des attributs ARIA : Les attributs ARIA (Accessible Rich Internet Applications) aident à rendre les interfaces plus compréhensibles pour les technologies d'assistance. Par exemple, role et aria-label peuvent être utilisés pour décrire les éléments de l'interface utilisateur.
  • S'assurer d'un bon contraste : Vérifiez que le contraste entre le texte et le fond est suffisant pour les utilisateurs malvoyants. Des outils comme Contrast Checker peuvent vous aider à mesurer cela.

Exemple de formulaire HTML :


                    <form action="/envoyer" method="POST">
                      <label for="nom">Nom:</label>
                      <input type="text" id="nom" name="nom" required>
            
                      <label for="email">Email:</label>
                      <input type="email" id="email" name="email" required>
            
                      <label for="message">Message:</label>
                      <textarea id="message" name="message" required></textarea>
            
                      <button type="submit">Envoyer</button>
                    </form>

Les formulaires en HTML

Les formulaires en HTML permettent aux utilisateurs d'interagir avec un site web en envoyant des données, comme des informations de contact ou des commentaires. Ils sont composés de différents éléments, tels que les champs de texte, les cases à cocher, les boutons radio et les boutons d'envoi. Chaque élément du formulaire est enveloppé dans une balise <form> qui définit comment et où les données doivent être envoyées.

Gestion des médias en HTML

  • Utiliser des formats d'images optimisés : Pour améliorer la vitesse de chargement, utilisez des formats modernes comme WebP ou AVIF pour les images. N'oubliez pas de spécifier les attributs width et height pour éviter le changement de mise en page lors du chargement.
  • Balise <video> : Pour ajouter des vidéos, utilisez la balise <video> et fournissez plusieurs formats de vidéos pour une compatibilité maximale entre navigateurs.

Exemple de gestion des médias en HTML :


                        <!-- Insertion d'une image avec balise alt et dimensions définies -->
                        <img src="image.webp" alt="Description de l'image" width="600" height="400" />
                    
                        <!-- Utilisation de la balise video pour intégrer une vidéo -->
                        <video controls width="600">
                        <source src="video.mp4" type="video/mp4">
                        <source src="video.webm" type="video/webm">
                        </video>
                    
                        <!-- Inclusion d'une balise audio pour jouer de l'audio -->
                        <audio controls>
                        <source src="audio.mp3" type="audio/mpeg">
                        <source src="audio.ogg" type="audio/ogg">
                        </audio>
